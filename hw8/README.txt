Music Genre Get Python Script

Introduction
This python script connects to a MySQL database, retrieves a list of music genres, prompts the user to select a genre, and then call a stored procedure to display songs in the selected genre.


Prerequisites:

- Python 3.x
- MySQL Server
- Required Python libraries: mysql-connector
- musicxuc database

Instructions:
1. Download database and python file to your local
2. Ensure your MySQL is running and have downloaded Python libraries
3. Enter your MySQL username and password
4. Script will display a list of available music genres, enter the desired genre when prompted
5. View the songs and exit the program
